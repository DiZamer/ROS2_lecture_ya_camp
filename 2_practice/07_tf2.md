# Практика: tf2 — дерево координат

## Цель

Создать Python-пакет `tf2_demo` с двумя узлами: `static_broadcaster` (публикует статический transform с фрейма `world` в фрейм `robot_base` со смещением x=1.0, y=2.0, z=0.0 через `StaticTransformBroadcaster`) и `tf_listener` (каждую секунду запрашивает transform `world → robot_base` через `Buffer.lookup_transform()` и выводит координаты). Проверить через `tf2_echo world robot_base` и `view_frames` (генерирует PDF с деревом координат).

## Предварительные требования

- Выполнены практики [01-06](01_workspace.md)
- Workspace `~/ros2_ws` создан и активирован

## Что получится

- Пакет `tf2_demo` с зависимостями `rclpy`, `geometry_msgs`, `tf2_ros`
- Узел `static_broadcaster` (класс `StaticFramePublisher`): создаёт `TransformStamped()` с `header.frame_id = 'world'`, `child_frame_id = 'robot_base'`, translation (1.0, 2.0, 0.0), rotation (0,0,0,1). Публикует один раз в `/tf_static` через `StaticTransformBroadcaster.sendTransform()`
- Узел `tf_listener` (класс `TransformListenerNode`): создаёт `Buffer()` (кэш transforms) и `TransformListener(buffer, self)`. По таймеру 1.0 с вызывает `buffer.lookup_transform('world', 'robot_base', time)` и выводит координаты
- Проверка: `ros2 run tf2_ros tf2_echo world robot_base` (живой поток transform), `ros2 run tf2_tools view_frames` (PDF с деревом `world → robot_base`)
- Расширение: второй transform `robot_base → sensor` (0.5, 0.0, 0.3), tf2 автоматически вычисляет `world → sensor` как сумму transforms

## Шаг 2. Код static transform broadcaster

Создать `tf2_demo/static_broadcaster.py`:

```python
from geometry_msgs.msg import TransformStamped  # Сообщение: frame_id, child, translation, rotation
import rclpy
from rclpy.node import Node
from tf2_ros import StaticTransformBroadcaster  # Публикует transform в /tf_static (один раз)


class StaticFramePublisher(Node):
    """Публикует статический (неизменный) transform world → robot_base один раз при запуске"""

    def __init__(self):
        super().__init__('static_frame_publisher')
        # StaticTransformBroadcaster — отправляет transform в топик /tf_static
        # Подписчики получают transform один раз и кэшируют навсегда
        self.broadcaster = StaticTransformBroadcaster(self)

        # TransformStamped — полное описание transform:
        #   header.frame_id — родительский фрейм (откуда)
        #   child_frame_id — дочерний фрейм (куда)
        #   translation — смещение в метрах (x, y, z)
        #   rotation — поворот в кватернионе (x, y, z, w)
        t = TransformStamped()
        t.header.stamp = self.get_clock().now().to_msg()
        t.header.frame_id = 'world'
        t.child_frame_id = 'robot_base'
        t.transform.translation.x = 1.0
        t.transform.translation.y = 2.0
        t.transform.translation.z = 0.0
        t.transform.rotation.w = 1.0    # Кватернион (0,0,0,1) — нет вращения

        self.broadcaster.sendTransform(t)  # Публикация — один вызов, топик /tf_static
        self.get_logger().info(
            'Static transform world → robot_base published')
        self.get_logger().info(
            'Translation: (1.0, 2.0, 0.0)')
```

## Шаг 3. Код tf listener

Создать `tf2_demo/tf_listener.py`:

```python
import rclpy
from rclpy.node import Node
from tf2_ros import Buffer, TransformListener  # Buffer — кэш transforms, TransformListener — подписка


class TransformListenerNode(Node):
    """Слушает transform-ы и каждую секунду запрашивает world → robot_base"""

    def __init__(self):
        super().__init__('tf_listener')
        # Buffer — хранит все transform-ы, которые приходят в /tf и /tf_static
        # Позволяет запрашивать transform между любыми двумя фреймами в любой момент
        self.buffer = Buffer()
        # TransformListener — подписывается на /tf и /tf_static, заполняет Buffer
        self.listener = TransformListener(self.buffer, self)
        self.timer = self.create_timer(1.0, self.lookup)  # Запрос каждую секунду

    def lookup(self):
        """Запрашивает transform world → robot_base и выводит координаты"""
        try:
            # lookup_transform(target_frame, source_frame, time)
            # Возвращает transform, который переводит точку из source в target
            # rclpy.time.Time() — последний доступный timestamp
            t = self.buffer.lookup_transform(
                'world', 'robot_base', rclpy.time.Time())
            self.get_logger().info(
                f'robot_base in world: '
                f'x={t.transform.translation.x:.2f}, '
                f'y={t.transform.translation.y:.2f}, '
                f'z={t.transform.translation.z:.2f}')
        except Exception as e:
            self.get_logger().warn(
                f'Transform not yet available: {e}')  # Первые секунды transform может ещё не прийти
```

## Шаг 4. Точки входа и сборка

Добавить `main()` в оба файла:

```python
def main(args=None):
    rclpy.init(args=args)
    node = StaticFramePublisher()   # или TransformListenerNode()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()
```

`setup.py` → `entry_points`:

```python
entry_points={
    'console_scripts': [
        'static_broadcaster = tf2_demo.static_broadcaster:main',
        'tf_listener = tf2_demo.tf_listener:main',
    ],
},
```

Сборка:

```bash
cd ~/ros2_ws
colcon build --packages-select tf2_demo
source install/setup.bash
```

## Шаг 7. Расширение: цепочка из трех frames

Добавить в `static_broadcaster.py` второй transform — `robot_base → sensor`:

```python
t2 = TransformStamped()
t2.header.stamp = self.get_clock().now().to_msg()
t2.header.frame_id = 'robot_base'
t2.child_frame_id = 'sensor'
t2.transform.translation.x = 0.5
t2.transform.translation.y = 0.0
t2.transform.translation.z = 0.3
t2.transform.rotation.w = 1.0

self.broadcaster.sendTransform(t2)
```

Пересобрать, перезапустить. Проверить:

```bash
ros2 run tf2_ros tf2_echo world sensor
# - Translation: [1.500, 2.000, 0.300]
```

tf2 автоматически вычислил положение `sensor` в `world`: сложил transform `world→robot_base` + `robot_base→sensor` = `world→sensor`.

```bash
ros2 run tf2_tools view_frames
# Теперь дерево: world → robot_base → sensor
```

## Типичные ошибки

| Симптом | Причина | Исправление |
| --- | --- | --- |
| `tf2_echo` пустой (ждет) | Transform не опубликован | Проверить `sendTransform()` и `spin()` |
| `lookup_transform` — исключение | Timestamp не совпадает | `rclpy.time.Time()` для последнего доступного |
| `view_frames` — пустой PDF | Нет запущенных узлов с transforms | Запустить все узлы, затем `view_frames` |
| Разные frame_id | Transform не находится | Регистр важен: `world` vs `World` |
| `ImportError: No module named 'tf2_ros'` | Зависимость не объявлена | Добавить `<depend>tf2_ros</depend>` в `package.xml` |
